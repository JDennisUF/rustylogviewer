This directory contains sample log filenames for fixture and pattern-matching tests.

Files included here represent two common cases:

- Stable Linux-style log names with rotated suffixes such as `syslog`, `auth.log`, and `syslog.1`
- Custom application logs that include the date in the filename, such as `app_20260425.log` or `app_0425.log`

The dated examples use April 25, 2026.
